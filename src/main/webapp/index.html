POST 방식 요청

POST 방식 요청은 form 을 이용해 동작합니다. GET은 파라미터를 URL로 전달하지만 POST는 HTTP 프로토콜 메시지 body에 저장하여 요청을 보냅니다. POST는 서버 데이터의 추가, 수정, 삭제와 같은 변경 작업에 주로 사용됨. 파라미터 값 인코딩 X->한글깨짐 , 요청값 인코딩 필수
